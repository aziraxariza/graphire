# Graphire 🚀
 
### Empowering Women in Tech Through Smart Opportunity Discovery

Graphire is a platform designed to help women in tech discover relevant opportunities in one place. From internships and hackathons to scholarships, jobs, mentorships, and tech events, Graphire intelligently organizes and connects opportunities using graph-based data relationships for faster and smarter discovery.

Built using GraphSQL and TigerGraph, the platform focuses on creating meaningful connections between users, skills, domains, and opportunities.

---

## ✨ Features

* 🔍 Smart opportunity discovery
* 🌐 Graph-powered relationship mapping
* 💼 Internships, jobs, hackathons & scholarships
* 🎯 Personalized recommendations
* ⚡ Fast querying using GraphSQL
* 📊 Structured and connected opportunity data
* 👩‍💻 Built with a focus on women in tech

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js
* **Database:** TigerGraph
* **Query Language:** GraphSQL
* **Version Control:** Git & GitHub

---

## 📌 How It Works

Graphire uses graph database technology to map relationships between:

* Users
* Skills
* Domains
* Opportunities
* Organizations

This allows the platform to provide more relevant and connected recommendations compared to traditional search systems.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/graphire.git
```

Navigate to the project directory:

```bash
cd graphire
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
